## 2. Plan Mode, Extended Thinking ✅

- `Shift-Tab` x2 to toggle thinking mode
- Press `Ctrl+G` to view and edit plan 
- `"Think hard"` to force extended thinking
- Press `Cmd+O` to display the thinking
